위키와 문서

GitHub Pages 전환 후 공개 사이트에서 유지할 문서 범위를 정리한 정적 페이지입니다.

정적 운영 원칙

브라우저는 `data/*.json` 파일만 읽습니다. 서버 route, 세션, 데이터베이스 연결, 인증 callback은 포함하지 않습니다.

공개 데이터 기준

요약 지표와 전략 설명처럼 공개 가능한 집계만 둡니다. 개인 식별 정보, 주문 상세, 계정 정보, API 키, DB 정보는 저장하지 않습니다.

종료된 동적 기능

로그인, 관리자 화면, 콘솔, 게시판 작성, 서버 API 직접 호출은 정적 사이트에서 제공하지 않습니다.
