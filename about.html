ABOUT MEDIAMAK

미디어막이라는 이름

MediaMak은 데이터와 시스템을 만들고, 그 과정을 공개 가능한 기록으로 정리하기 위한 기술블로그 이름입니다.

NAME

Media + Mak

Media는 글, 리포트, 문서, 출판물처럼 공개 가능한 기록을 뜻합니다. Mak은 만들고 쌓아 올리는 작업을 가볍게 줄인 이름으로 사용합니다.

SCOPE

기록하는 대상

자동매매 시스템 개발, 데이터 수집과 정제, 백테스트 검증, 실거래 운영에서 얻은 기술적 교훈을 다룹니다.

BOUNDARY

이 사이트가 하지 않는 것

  • 투자 권유, 자문, 수익 보장을 목적으로 하지 않습니다.
  • 실시간 매매 신호나 따라 하기용 주문 정보를 제공하지 않습니다.
  • 계정, 주문 상세, API 키, 관리자 로그처럼 민감한 정보는 공개하지 않습니다.

READING PATH

처음 읽는 순서

01최신 글최근 데이터 인사이트와 개발 기록을 먼저 봅니다.
02시스템 흐름수집기, DB, 전략 러너, 정적 공개 구조를 확인합니다.
03검증 기록백테스트, 운영 집계, 이전 기록의 공개 자료를 확인합니다.
04책 목차장기적으로 정리할 원고 흐름을 봅니다.
